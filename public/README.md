# El Djamila Salon Application

A modern web application for a hair salon with authentication, booking system, live sessions, and admin panel.

## Features

- **User Authentication**: Login and registration system
- **Admin Panel**: Manage services, users, and view statistics
- **Service Booking**: Browse and book hair services
- **Live Sessions**: Watch and participate in live tutorials
- **Contests**: Participate in salon contests
- **Wallet System**: Manage balance and points
- **Multi-language**: French, English, and Arabic support
